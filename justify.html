justify-content e align-items

O comando justify-content é usado para você alinhar seus itens no eixo principal (por padrão horizontal), enquanto o comando align-items é usado para alinhar seus itens no eixo transversal (por padrão vertical).
Nos exemplos abaixo estaremos usando os eixos padrão.

justify-content

justify-content: center

Quando aplicamos essa característica no CSS, centralizamos os itens no meio da linha se o flex-direction for row/row-reverse ou centraliza verticalmente se o flex-direction for column/column-reverse.

teste123

justify-content: flex-start

Quando aplicamos essa característica no CSS, ele coloca o objeto no começo da linha se for flex-direction row/row-reverse ou coloca o objeto no topo do container para flex-direction column/column-reverse.

teste123

justify-content: flex-end

Quando aplicamos essa característica no CSS, colocamos o objeto no final da linha para flex-direction row/row-reverse ou coloca o objeto na parte inferior da página se for flex-direction column/column-reverse.

align-items

Quando aplicamos essa característica no CSS, fazemos ele se comportar igual stretch do flexbox, esticando os itens para preencher o container.
