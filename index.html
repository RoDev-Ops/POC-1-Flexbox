Ativar Flexbox

Para ativar o flexbox, basta adicionar no css o comando "display: flex;" Por padrão o flex directions é setado para row, mais sobre flex-direction a seguir.

flex-direction: row;

Ao usar flex directions column você alinha seus itens na horizontal, seguindo a ordem que foram colocados no html, como o examplo abaixo:

justify-content e align-items

O comando justify-content é usado para você alinhar seus itens no eixo principal setado pelo flex-direction, enquanto o comando align-items é usado para alinhar seus itens no eixo transversal.

justify-content + flex-direction: row

justify-content: center

Quando aplicamos essa característica no css e temos flex-direction row, centralizamos os objetos no meio da linha
